Power Supply Waveform Tools

A unified web-based toolkit designed to visualize and edit test scripts for DC Power Supplies. This project supports both Chroma 62000P (SCPI Sequence) and GW Instek PSW (CSV) formats, enabling engineers to verify automotive test profiles (e.g., ISO-16750-2) directly in the browser.
